## Project Setup

```sh
git clone git@github.com:Anemoiaa/vue-joke.git
```

```sh
cd vue-joke
```

```sh
rename .env-example to .env and configure it
```

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
